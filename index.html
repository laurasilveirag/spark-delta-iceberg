🛒 E-commerce Lakehouse - SATC

Projeto desenvolvido para a disciplina de Arquitetura de Dados da SATC, explorando tecnologias modernas de armazenamento e processamento de dados.

📌 Sobre o Projeto

Simulamos o ambiente de dados de um E-commerce com duas fontes:

  • clientes.csv — cadastro de clientes com nome e estado
  • vendas.csv — registro de vendas com valor, status e data

O objetivo foi demonstrar como Delta Lake e Apache Iceberg funcionam dentro do Apache Spark, realizando INSERT, UPDATE e DELETE de forma confiável.
